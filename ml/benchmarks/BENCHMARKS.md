# Benchmarks

No benchmarks published yet — this file will hold the versioned, dated history of Vestrix's detection accuracy once a baseline model exists.

## Format for future entries

Each entry must include:
- Date
- Model version / config hash
- Dataset used (and whether it's public / where to get it)
- Methodology (train/test split strategy, cross-validation approach)
- Leave-one-room-out and leave-one-device-out results, not just in-distribution accuracy
- False-positive and false-negative rates
- Confusion matrix / per-environment breakdown
- Known limitations for this specific model version

Never overwrite a previous entry — append new dated entries so the improvement (or regression) history stays visible.
